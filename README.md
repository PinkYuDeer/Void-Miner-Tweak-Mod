注意：
安装mod后会基于你当前的游戏语言，向GregTech.lang写入翻译条目，而写入后无法自动移除条目。
因此请先设置好游戏语言再安装此mod。
如果在运行过mod之后想更改语言，且启用了UseThisFileAsLanguageFile=true，你需要删除GregTech.lang，或者恢复安装mod前的GregTech.lang的备份（若有）。
